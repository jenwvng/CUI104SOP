# Safe handling and disposal of chemicals #
It is crucial to safely handle and properly dispose of all chemicals. Gloves must be worn when handling chemicals and all handling must be done in the fume hood. Green heavy duty rubber gloves must be worn when directly touching acid when acid washing glassware. Do not store chemicals in the fume hood.

Any acidified samples from the NPOC must be discarded into an acid waste container instead of down the drain. It can be directly poured into the waste container.

Any broken glass must be disposed of inside the broken glass/sharps container. 

If there is a spill, the spill must be contained using a spill kit if necessary. 

For the proper labeling and disposal of waste containers, please refer to the SOP on https://www.torontomu.ca/facilities-management-development/environmental-health-safety/laboratory-safety/chemical-safety/#!accordion-1504294341862-disposing-of-hazardous-material
